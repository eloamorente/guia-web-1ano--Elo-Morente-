O que é HTML, CSS, JS?

HTML

Estrutura o conteúdo da página web.

<h1>Título</h1>
Use para estruturar textos, imagens e links.

CSS

Define o estilo visual da página.

h1 { color: blue; }
Use para cores, fontes e layout.

JS

Adiciona interatividade e lógica.

document.querySelector('h1').onclick = ...
Use para animações, validações e dinâmicas.
